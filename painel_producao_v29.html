# Descrição
📋

Importe um roteiro para começar a rastrear as partes.

Configurar estrutura de pastas

💡 Como escrever o caminho corretamente:
  • Use o formato completo do Windows, com letra da unidade: C:\Projetos\MeuCanal\EP03
  • Evite espaços no nome das pastas do caminho (ex: use pastas_video_5 em vez de pastas video 5). O script já coloca aspas automaticamente, mas nomes sem espaço são mais seguros de digitar, copiar e usar depois em outros programas.
  • Evite também acentos e caracteres especiais (ç, ã, /, :, *, ?, ", <, >, |) — eles podem causar erro na hora de criar a pasta.
  • Use barra invertida \ (não /) para separar as pastas — é o padrão do Windows.
  • Não é preciso a pasta já existir: o .bat cria toda a estrutura, inclusive pastas intermediárias que ainda não existem.
Script gerado — pastas
💡 Salve o arquivo como criar_pastas.bat, clique com botão direito → "Executar como administrador". As pastas serão criadas instantaneamente.

Fluxo de trabalho recomendado

  1. Importe o roteiro — Cole o texto numerado do Word/Google Docs na aba "Importar roteiro". Cada item numerado vira uma parte.
  2. Gere as pastas — Na aba "Script de pastas", configure o caminho e baixe o .bat. Execute uma vez — todas as pastas são criadas com o mesmo nome que aparece no painel.
  3. Busque os materiais — Abra o Envato/Storyblocks, encontre o vídeo/foto para cada parte. Cole o link na coluna "Link do material" da respectiva parte.
  4. Marque como "Aguardando download" — Ao colar o link, o status muda automaticamente. Agora você pode continuar buscando outros materiais.
  5. Volte para baixar — Filtre por "Aguardando download". Veja exatamente quais precisam ser baixados e para qual pasta cada um vai.
  6. Avance o status — Após baixar e organizar na pasta, clique "→" ou use o dropdown para marcar como "Baixado".
  7. Salve o progresso — Clique em "💾 Salvar" a qualquer momento. O arquivo HTML salva os dados localmente no seu navegador.

Dicas

  • Os nomes das pastas são gerados automaticamente no formato 001_Nome_da_parte — igual no painel e no Windows.
  • Use "Exportar CSV" para ter uma planilha com todo o histórico do projeto.
  • Você pode importar vários roteiros e salvar projetos diferentes — o painel guarda o estado no navegador.
  • Funciona offline — não precisa de internet depois de abrir o arquivo.
